# 词库预处理工具集 / Vocabulary Preprocessing Toolkit

本目录包含用于 Galaxy Vocab 项目的 Python 词库处理工具，旨在实现数据清洗与格式规范化。

## 工具列表 / Tools

### 1. `process_json.py` (词库整合工具)
- **作用**：将分布在多个分类文件中的 JSON 词库合并为单一的全局词库文件。
- **逻辑**：自动遍历输入目录，提取有效单词条目，并进行格式校验后统一输出。

### 2. `extractor.py` (文本提取工具)
- **作用**：从原始文本或 PDF 导出的文本流中，通过正则表达式提取单词及其对应的中文释义。
- **逻辑**：支持自定义匹配模式，自动过滤非词汇内容，实现半自动化词库构建。

## 使用说明 / Usage
1. 确保已安装 Python 3.x 环境。
2. 所有脚本均已配置为**相对路径**，请在 `scripts/` 目录下运行。
3. 示例命令：
   ```bash
   python process_json.py
   ```

---
*注：这些工具主要用于开发阶段的数据准备，不直接参与小程序线上运行。*
